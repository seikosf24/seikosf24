## はじめに

## どこに何があるか、について
全体のCSSはstyle.css、全体のJSはscript.jsに入っています。他は見ての通りです。
- 開催情報(info)
- 企画案内(booth)
- 講堂企画(booth)
- ブログ(blog)
- よくある質問(faq)

- 素材系(assets)
- 写真(image)
